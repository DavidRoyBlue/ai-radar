DeepZero is an autonomous vulnerability-research framework that parses, decompiles, and analyzes Windows kernel drivers for exploitable conditions. The interesting bits aren't the security domain — they're the long-horizon orchestration: how it queues drivers, parallelizes analysis, decides when to escalate to deeper symbolic checks, and persists findings. Same architecture applies to any "find-things-in-a-large-corpus" agent task, including the radar itself.
