Open-source MCP servers from AWS Labs covering EC2, S3, Lambda, CloudFormation, and growing. Notable because (a) it's first-party so the API coverage tends to be authoritative and (b) AWS Labs is shipping MCP at a brisk cadence, which is a leading indicator that cloud-vendor-supplied MCP is becoming standard infrastructure rather than a hobby. Adopt directly rather than wrapping the AWS SDK yourself in agent skills.
