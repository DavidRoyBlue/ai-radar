browser-use is a mature library that makes websites usable by agents — accessibility-tree-style page representation, click/type primitives, automatic captcha and login handling. Established enough that it's table stakes; track for new browser-side primitives (multi-tab orchestration, login state management) that change what's possible for autonomous web agents. Comparison point if the reader is evaluating Chrome DevTools MCP or BrowserMCP.
