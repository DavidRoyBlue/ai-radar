skill-optimizer diagnoses and optimizes Agent Skills (SKILL.md) using real session data plus research-backed static analysis. The feedback loop — instrument skill invocation, identify weak spots, suggest edits — is the right shape for treating skills as code that gets refactored over time rather than write-once-and-forget. Even if you don't adopt the tool, the static-analysis checks (which YAML fields, what description shape, what triggers a model to invoke) are a useful skill-authoring checklist.
