A Claude Code skill that gives the model browser-automation via Playwright — but instead of pre-encoding scripts, it's model-invoked: Claude autonomously writes and runs Playwright code per task. This is a useful template for any skill that wraps a flexible underlying tool: rather than expose a fixed tool surface, expose the *capability* and let the model decide what code to run. Read the SKILL.md to see how the safety/scope boundaries are described.
