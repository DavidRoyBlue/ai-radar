mcp-use is a fullstack framework for building both MCP apps (clients) and MCP servers. Targets the gap that the official SDKs leave open — opinionated scaffolding, dev server, hot reload, examples. Worth a look when starting a fresh MCP project, but the radar reader should still default to the official SDK if a minimal server is enough.
