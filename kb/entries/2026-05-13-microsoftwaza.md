microsoft/waza is a CLI and framework for creating, testing, measuring, and improving Agent Skills. Notable because it treats skills as engineering artifacts with measurable quality — not just markdown to drop into a directory. Read the metrics it computes (effectiveness, redundancy, conflict) as inspiration for a skill-portfolio audit step you can add to your own development workflow.
