obra/superpowers is an opinionated agentic-skills framework and software-development methodology that bakes TDD, brainstorming, plan execution, debugging, and verification into a coherent skill library that Claude Code invokes automatically. The interesting bit is the *discipline contract* between the skills — `using-superpowers` requires invocation before anything else, `verification-before-completion` blocks unverified completion claims, and skills compose without conflict. Study how the priority/red-flag tables and DOT-graph workflows are encoded as plain markdown the model actually follows; the same pattern is directly transferable to the radar agent's own behavior contract.
