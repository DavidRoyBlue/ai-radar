oraios/serena is an MCP toolkit that gives coding agents semantic retrieval and editing over a codebase — effectively "an IDE for your agent". Where most coding-agent setups rely on grep + read-file, Serena layers Language Server Protocol-driven symbol queries on top so the agent can navigate by *meaning* rather than text. Drop this MCP into Claude Code and you typically get smaller, more accurate diffs because the agent understands what's a definition vs a reference; well worth a 10-minute trial.
